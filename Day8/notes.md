# Day8 Linux 软件包管理

## 一、学习目标

Linux 日常使用中，经常需要安装、更新和删除软件。

Ubuntu 使用的软件管理工具：

> **APT（Advanced Package Tool）**

今天学习：

-   软件源（Repository）
-   apt 基本使用
-   软件安装
-   软件删除
-   软件搜索
-   系统更新

------------------------------------------------------------------------

## 二、Linux 软件包概念

### Windows

通常安装：

-   `.exe`
-   `.msi`

### Linux（Ubuntu）

使用：

-   `.deb`

APT 可以自动完成：

-   下载软件
-   安装软件
-   自动解决依赖
-   更新版本

------------------------------------------------------------------------

## 三、软件源（Repository）

查看软件源：

``` bash
cat /etc/apt/sources.list
```

新版 Ubuntu：

``` bash
ls /etc/apt/sources.list.d/
```

------------------------------------------------------------------------

## 四、更新软件列表

``` bash
sudo apt update
```

作用：刷新软件列表，不会安装更新。

------------------------------------------------------------------------

## 五、升级软件

``` bash
sudo apt upgrade
```

作用：安装已经下载到软件列表中的更新。

  命令          作用
  ------------- --------------
  apt update    更新软件列表
  apt upgrade   安装更新

------------------------------------------------------------------------

## 六、安装软件

``` bash
sudo apt install tree
```

查看目录：

``` bash
tree
```

------------------------------------------------------------------------

## 七、删除软件

普通删除：

``` bash
sudo apt remove tree
```

彻底删除：

``` bash
sudo apt purge tree
```

------------------------------------------------------------------------

## 八、搜索软件

``` bash
apt search vim
```

查看已安装：

``` bash
apt list --installed
```

------------------------------------------------------------------------

## 九、总结

最常用命令：

``` bash
sudo apt update
sudo apt upgrade
sudo apt install 软件名
```
