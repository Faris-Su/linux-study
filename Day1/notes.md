# Day 1 Linux 入门

## 学习目标

完成 Ubuntu 虚拟机环境搭建，开始熟悉 Linux 基本操作。

环境： - Ubuntu 26.04 LTS - VMware Workstation - Windows 11

## Linux 基本认识

Linux 是一种开源操作系统，广泛应用于： - 服务器 - 云计算 - 网络设备 -
嵌入式系统

特点： - 命令行操作为主 - 配置文件透明 - 权限管理严格

## 文件系统

常见目录：

  目录    作用
  ------- ----------------
  /home   用户文件
  /etc    系统配置
  /var    日志和运行数据
  /usr    系统程序
  /tmp    临时文件

## 常用命令

查看路径： pwd

查看目录： ls

详细查看： ls -l

切换目录： cd 目录名 cd .. cd \~

## 总结

熟悉 Linux 环境，为后续学习文件管理、权限和网络配置打基础。
